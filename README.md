# TChemistry

## Info
A minecraft mod about tech and chemistry. From making new alloys to automating reactors.

## What makes up the mod
* New Alloys
* Chemicals
* Infuse alloys with chemicals to make better alloys
* Multiblocks
* Tiered progression:
   * FG Age
   * Steel Age
   * Semi-Nuclear Age
   * Nuclear Age
   * Nuclear & Beyond Age
